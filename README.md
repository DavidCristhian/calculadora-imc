# 🧮 Calculadora de IMC

Este projeto foi desenvolvido para colocar em prática conceitos fundamentais de desenvolvimento web. O principal objetivo foi aplicar o aprendizado em manipulação do DOM (Document Object Model) para criar uma interface interativa e responsiva.

## 🚀 Tecnologias Utilizadas

* **HTML5**: Estruturação semântica da página.
* **CSS3**: Estilização moderna e layout responsivo.
* **JavaScript (Vanilla)**: Lógica do cálculo de IMC e manipulação dos elementos da tela.

## 🧠 Aprendizados Coletados

* Captura de dados inseridos pelo usuário em formulários.
* Eventos de clique e validação simples de campos de entrada.
* Alteração dinâmica de classes e textos na tela de forma interativa.
